# Sunset Chat
## 1. Descripcion del proyecto
Este proyecto consiste en la **maquetacion en html y css** de una interfaz web inspirada en WhatsApp, llamada **Sunset Chat**. 
Se reproduce la **vista principal del panel de chats**, que incluye un sidebar con una lista de conversaciones y un panel principal con la conversacion activa en ese momento.
El proyecto fue desarrollado utilizando HTML Y CSS para su estructura y diseño visual.git

## 4. Decisiones de diseño 
- El diseño está basado en WhatsApp y con la base creada durante las clases, pero intente crear una **identidad visual propia** utilizando una paleta de colores en tonos naranjas y azules inspirados en el atardecer pero que mantengan un buen contraste visual.  
- Utilice la tipografía **Roboto**, importada desde Google Fonts, que fue la seleccionada para el proyecto original y resulto ser clara y legible para el chat.
- La estructura general la organice con **CSS Grid** para dividir el layout en dos columnas principales (sidebar y el panel de conversacion). Considere que esta herramienta era la adecuada para definir bien la distribucion de los elementos y adapatarlos a los distintos tamanos de pantalla.
- Para la alineación y distribución de los elementos internos dentro de cada seccion utilice **Flexbox**. Esto me permitio mantener el codigo ordenado en encabezados, en el panel de chats, el panel de mensajes  y el footer. Asi los elementos fueron alineados y espaciados segun el diseno original que estabamos tomando de referencia.
- Se definieron **variables CSS** para colores y tipografía, facilitando el mantenimiento del código.  
- Se implementó un diseño **responsive**, ocultando el sidebar en pantallas pequeñas.
## 5. Trabajo con Git 
Durante el desarrollo del proyecto trabajé de forma progresiva, realizando **commits frecuentes y descriptivos** para registrar cada avance.
Comencé con la estructura inicial y el HTML base, y luego fui agregando los estilos de las distintas secciones de la interfaz. A lo largo del proceso incorporé **variables de colores**, la **tipografía**, mejoras visuales, **diseño responsive** y **assets** como el ícono del proyecto.
Esto me permitió mantener el código ordenado y llevar un control del progreso del trabajo.
